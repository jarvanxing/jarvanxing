实战经验

  1. 在shouldComponentUpdate中优化组件减少reRender
  2. propTypes, defaultProps不要懒的去写，别人通过你的propTypes很容易理解组件，也容易debug
  3. immutable.js 一定要注意数据的变异
  4. 组件间的数据通信(eventmitter, context, flux)
  5. 无状态组件
  6. 带widnow/dom的操作抽离出来, 方便同构
  7. 图标管理(svg)
  8. code split 按需加载
  9. seo via nodejs - prerender

缺点

  1. 不支持ie9以下
  2. 周边插件不稳定，更新太频繁
